## 깃허브 꿀팁
같이 페어로 작성한 코드를 한 사람이 커밋할 때 페어의 이름이 빠지는 것이 서운하다면 Co-authored-by를 커밋메시지 설명에 추가하면 됩니다. 단, 꼭 두 줄을 비워야합니다!

```
$ git commit -m "Refactor usability tests.
>
>
Co-authored-by: name <name@example.com>
```
